# rrdoctor Streamlit demo

This tiny web demo lets a user paste a public GitHub repository URL or click a sample repository, shallow-clones it into a temporary directory, runs the static `rrdoctor scan` command, and displays the reproducibility score plus the top findings. It is designed for Streamlit Community Cloud or Hugging Face Spaces; install `demo/requirements.txt`, run `streamlit run demo/app.py`, and make sure the runtime has `git` available. On Streamlit Community Cloud, select a current supported Python runtime in the app's advanced settings and redeploy after dependency changes; when `demo/requirements.txt` pins a just-published rrdoctor version, redeploy after PyPI lists that version as available. The public app may also show a wake-up screen after inactivity before the rrdoctor input box appears.

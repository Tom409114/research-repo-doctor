# Roadmap

## v0.1.0

- Deterministic CLI scans.
- Markdown, JSON, and experimental SARIF-compatible reports.
- GitHub Action integration.
- Public docs and contribution workflow.
- Issue templates for bugs, false positives, false negatives, rule requests, and scan cases.
- Self-scan report published under `examples/reports/`.

## v0.2.0

- More robust SARIF output and code scanning docs.
- Additional language ecosystem checks for R, Julia, MATLAB, and JavaScript research repos.
- More fixture-driven false positive reduction.
- Optional rule packs for ML, computational social science, and bioinformatics.
- Better notebook path and output hygiene heuristics.
- Zenodo, DOI, and arXiv metadata detection improvements.

## v0.3.0

- Better report diffing for pull requests.
- Maintainer tools for issue triage and rule request clustering.
- Expanded examples from real public test repositories, with permission and links.
- Docs website deployment and versioned documentation.
- Workflow detection for Snakemake, Nextflow, and other research pipelines.

## Good first issues

- Add R language environment detection.
- Add Julia environment detection.
- Improve notebook path detection.
- Add Zenodo/DOI detection.
- Add Snakemake/Nextflow workflow detection.
- Add docs website deployment.

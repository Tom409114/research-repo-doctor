## Short PR Description

Add Research Repo Doctor to the Python development resources section.

Research Repo Doctor is a deterministic, local-first CLI for auditing research repositories before artifact review, release, or handoff to a coding agent. It checks for common reproducibility gaps such as missing environment metadata, weak data provenance, stale notebook outputs, and unreproducible randomness, and it can generate non-destructive starter fixes.

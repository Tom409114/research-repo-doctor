## Short PR description

This adds Research Repo Doctor (`rrdoctor`) to the Development Resources section. `rrdoctor` is a deterministic, local-first CLI and GitHub Action for checking research repositories for reproducibility readiness, including environment metadata, data provenance, notebooks, citation metadata, governance, CI, and experiment entrypoints. It does not require an API key or hosted service.
